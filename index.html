Programmation LOGO

Bienvenue au cours de programmation Logo! Logo est un langage de programmation qui utilise des commandes graphiques pour dessiner des formes à l'écran. Il est souvent utilisé pour enseigner la programmation aux enfants car il est facile à apprendre et amusant à utiliser.

Commençons par le début

Le logo utilise une tortue pour dessiner des formes à l'écran. La tortue peut avancer, reculer, tourner à gauche ou à droite, et même dessiner des formes.



Environnement LOGO

  • Barre de menus : permet d’accéder aux fonctionnalités permises par la fenêtre de Logo.Net. (Enregistrement, fermeture, paramétrage, Editeur, Affichage, …)

  • Barre d’outils : contient des icônes permettant d’exécuter quelques options de la barre de menus (Enregistrer, Editeur, Verrouiller, …).

  • Espace de dessin : Dans lequel la tortue dessine lorsqu’elle se déplace.

  • La tortue : C’est elle qui exécute les commandes.

  • Console : C’est dans cette zone qu’on saisit les commandes pour piloter la tortue.

  • Historique : L’historique de toutes les instructions saisies et tenues dans cette liste.

  • Barre d’état : affiche un ensemble d’informations sur la tortue (les coordonnées, l’orientation, les dimensions de la zone graphique, …).



Activité :

Donner le chemin pour que Salwa arrive à l'école en utilisant les ordres :
  • - Avancer
  • - Tourne droite
  • - Tourne a gouche

Programmer en LOGO

Pour demander à la tortue de tracer un dessin, vous devez saisir des commandes dans la console et de cliquer sur la touche Entrer du Clavier.

1) Les commandes de déplacement :

  • - AV x : Avance de X pas sur sa direction
  • - RE x : Recule de X Pas sur sa direction
  • - ORIGINE : Repositionne la tortue à l’origine
  • - FPOS [x y] : Positionne la tortue sur le point des coordonnées (x,y)

2) Les primitives du langage Logo :

  • - LC : Lève Crayon
  • - BC : Baisser Crayon
  • - VE : Vider l’Écran
  • - VC : Vide la console

3) Les commandes d’orientation :

  • - TD x : Tourne de X degrés à Droite
  • - TG x : Tourne de X degrés à Gauche
  • - Fcap n : Oriente la tortue sur l’angle n°

4) Taille et type du trait :

    Le crayon de la tortue peut être paramétré au niveau de la taille et du type en utilisant les commandes indiquées dans le tableau suivant :

  • - FEPAIS x : l’épaisseur du trait du crayon à la valeur x ; (1 < x < 30)
  • - FTRAIT x : le type de trait :
    • Si x = 0 : trait continu
    • Si x ≠ 0 : trait pointillé.

Les boucles :

Une boucle c’est une structure de contrôle permettant de répéter le même bloc d’instruction plusieurs fois.

Syntaxe : REPETE n [bloc d’instructions]

REPETE : est la primitive à utiliser pour demander à la tortue d’exécuter un bloc d’instruction un nombre de fois bien défini.


** Exemple pour tracer un carré : **

REPETE 4 [AV 60 TD 90]



Les procédures :

Une procédure un ensemble d’instructions destiné à faire accomplir une tâche précis. Pour créer une procédure, on distingue deux étapes :

Étape de définition :

POUR NomProcédure bloc d’instructions FIN

Étape d'appel :

Pour exécuter une procédure, on a deux possibilités :

1) Saisir son nom dans la console.

2) Sélectionner son nom dans la boite d’outil « Procédures ».


** Exemple pour tracer un carré : **


POUR Carrécqt AV50 TD90 AV50 TD90 AV50 TD90 AV50 FIN
